왜 이거모향? 을 만들게 되었나?

향수에 대해 잘 모르는 사람들은 원하는 향을 분위기에 비유해서 떠올린다.
사람들이 원하는 분위기를 대부분의 향을 나누는 기준별로 선정해서
향수에 더 쉽게 접근할 수 있는 앱을 만들게 되었다.

앱 사용 방법

    1. 감성적인 분위기로 먼저 접근한다
    2. 마음에 드는 분위기를 선택한 후 해당 향수 카테고리로 이동
    3. 다양한 향수의 특징들을 확인하고
    4. 원하는 향수는 찜목록에 추가할 수도 있다
